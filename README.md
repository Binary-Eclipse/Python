# 🧠 The Core Python Mental Model

The most important concept in Python is:

```text
NAME
  ↓
REFERENCE
  ↓
OBJECT
  ↓
TYPE + VALUE + IDENTITY
```

Example:

```python
x = [1, 2, 3]
```

Conceptually:

```text
x
│
└──────────────► List Object
                  │
                  ├── Type: list
                  ├── Identity: memory identity
                  └── Value: [1, 2, 3]
```

Understanding this explains:

* Mutable vs immutable
* Assignment
* Copying
* Function arguments
* Lists inside tuples
* Hashability
* Memory behavior
* Object identity

# ⚙️ Python Execution Model

```text
Python Source Code
        ↓
      Lexer
        ↓
      Parser
        ↓
       AST
        ↓
    Compiler
        ↓
     Bytecode
        ↓
 Python Virtual Machine
        ↓
   Python Objects
        ↓
     Program Output
```

Example:

```python
x = 10
print(x)
```

Conceptually:

```text
Source Code
    ↓
Parse
    ↓
Compile
    ↓
Bytecode
    ↓
PVM
    ↓
Create Integer Object
    ↓
Bind x to Object
    ↓
Call print()
```
